Safe Emergency Recovery Experiment — 研究ドキュメント Wiki

本リポジトリは、Docker Compose ベースの意図的に破壊可能な標的環境に対して LLM が応急復旧を試みる研究評価基盤である。 この Wiki は、リポジトリ内の Markdown ドキュメントを横断的に閲覧するための入口ページである。

左サイドバーから直接ページを開くか、下のカードからカテゴリ別に辿ること。 各ページは docs/ 配下のソース Markdown を pandoc で HTML 化したものであり、 内容を更新する場合は元の .md を編集して docs/wiki/_build.sh を再実行すること。
